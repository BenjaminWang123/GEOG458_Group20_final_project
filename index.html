01 · Project Overview

Why this project matters

Commercial districts are shaped by uneven patterns of movement, attraction, and visitation. Looking only at static locations does not fully explain how urban commercial life functions. This project uses spatial and temporal data to reveal where activity concentrates, when it changes, what types of places attract visitors, and how people move between places.

The website is designed as an interactive story platform: each page focuses on one theme while maintaining a consistent visual structure for comparison and exploration.
